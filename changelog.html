Changelog

Version 1.5.1
Released on March 29, 2025
  • Added bonus support to scrimmage mode with proper state management, timing, and score tracking
  • Improved game state handling during transitions between tossups and bonuses
  • Enhanced error handling and cleanup during game termination
  • Updated UI to show bonus settings and reading speed in game information
  • Fixed various bugs:
    • Fixed duplicate method definition in ScrimmageGame class
    • Fixed issue with bonus audio playback state management
    • Fixed handling of interrupted bonus cycles
    • Fixed player buzz validation
    • Fixed context handling for bot commands in study mode
    • Fixed typo in tossup message placeholder
    • Fixed error handling in voice client disconnection
Version 1.5.0
Released on March 15, 2025
  • Created new scrimmage mode, which allows users to read questions (currently only tossups for now, bonuses will be coming later) from a specific packet from a specific set, with support for custom packets planned.
  • The ability to create teams is now available in both tossup and scrimmages, which keeps track of total team stats.
  • Fixed issue where voice client disconnect prevented the end-to-game sessions, users now will be able to run '!end' and not face an error.
  • Fixed issue with text updating, where the question text would update between two separate discord messages.
Version 1.4.2
Released on February 17, 2025
  • Added ability to change reading speed.
  • Removed the power mark from being read aloud or printed in the text.
Version 1.4.0
Released on January 31, 2025
  • Added study mode (the bot automatically moves onto the next tossup after the previous one finishes).
  • Added the ability to select subcategories for tossups.
  • Buzzing in now has a ten-second timeout (so can't stall on buzzes anymore).
  • Changed voice to a more humanlike model.
Version 1.3.2
Released on January 19, 2025
  • Added multiplayer functionality, multiple people can play and compete on the same tossups.
  • Added the ability to toggle being able to buzz multiple times on the same question.
  • Fixed bug where all correct buzzes were treated as powers.
  • Added more visuals to the category selection.
Version 1.3.1
Released on January 15, 2025
  • Added delay to buttons from being interacted with until they are fully loaded.
  • Bot will now print text in sync with the read tossup.
  • Improved labels for difficulties.
  • Player score now broken down into powers, tens, and negs, along with the total score.
Version 1.3.0
Released on December 13, 2024
  • Created fleshed out UI system for game setup.
  • Added a buzz button to ease use for mobile users.
  • Added buttons at the end of each tossup to run different commands.
Version 1.2.0
Released on October 10, 2024
  • Modified buzzing and answer system to accept plaintext answers.
  • Redid file system so multiple game sessions can be run.
  • Added logging.
Version 1.0.0
Released on August 28, 2024
  • Able to read aloud tossups and accept buzzes and answers.
  • All command-based, including buzzes and answers.
  • Starting games require categories and difficulties to be written into the command.
