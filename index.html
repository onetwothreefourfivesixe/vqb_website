Welcome to QBVReader!

Improve your Quizbowl skills with this audio powered practice app. Play and compete with your teammates in the same sessions in a Discord server.

About QBVReader

QBVReader is a Quizbowl practice app that allows you to play audio questions or run scrimmages in a Discord server. Unlike existing apps, QBVReader supports not only tossups, but also bonuses and even whole packets for you or your team to run scrimmages. In fact, you can even play against multiple teams in the same session. If this sounds interesting or helpful to you or your team, please check out the installation page!

If you want to try the app first, try out the web version here or join the QBVReader Discord server.

This is a passion project of Edward Lin, and I hope you find it useful. This is very much still in development, so please report any bugs you find or features you would like to see implemented.
